perplexity ---------------
<div class="marquee-container">
  <div class="marquee-content">
    Ceci est un texte défilant créé avec CSS, remplaçant l'ancien élément HTML marquee.
  </div>
</div>

.marquee-container {
  width: 100%;
  overflow: hidden;           /* masque ce qui dépasse */
  background-color: #f0f0f0;
  border: 1px solid #ccc;
}

.marquee-content {
  display: inline-block;
  white-space: nowrap;        /* empêche le retour à la ligne */
  padding-left: 100%;         /* place le texte hors écran à droite */
  animation: marquee 15s linear infinite;
}

@keyframes marquee {
  0% {
    transform: translateX(0);        /* départ hors écran à droite */
  }
  100% {
    transform: translateX(-100%);    /* fin hors écran à gauche */
  }
}
